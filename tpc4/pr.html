Abstract:

Este trabalho tinha como objetivo o desenvolvimento de um servidor web em NodeJS, de modo a responder a vários pedidos HTTP, com a informação relativa a um arquivo de músicas.

Como o arquivo sonoro estava em formato XML, foi feito em aula uma stylesheet XSLT que dividisse as músicas em ficheiros XML separados.

De seguida foi implementado o servidor de modo a que respondesse a pedidos GET, para visualização das informações de uma das 448 músicas, utilizando a rota 'musica/num'.

Para representar a informação de cada música em HTML, foi necessário criar uma stylesheet XSLT ' doc2html.xsl' que aplicasse transformações ao ficheiro XML da música, aquando do respetivo pedido ao servidor.
