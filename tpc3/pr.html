Abstract:

Este TPC tinha como objetivo a criação de duas stylesheet's em formato XSLT, em que cada uma seria usada para aplicar transformações a dois ficheiros XML independentes. Estas transformações permitem gerar os ficheiros HTML correspondentes, tendo em conta os templates definidos para cada tag do ficheiro XML.

O primeiro ficheiro XML a ser transformado, corresponde ao project record, pr.xml, que foi especificado em aula, necessário para documentar todos os trabalhos da disciplina. A transformação com a stylesheet pr.xsl resultou nesta única página HTML.

De seguida, foi criada uma stylesheet que realizasse transformações ao ficheiro XML sobre os arqueossítios, arq.xml. Como tal, as transformações definidas com a stylesheet arq.xsl, permitiram gerar um conjunto de páginas HTML que originaram um website, contendo toda a informação guardada no ficheiro XML.
