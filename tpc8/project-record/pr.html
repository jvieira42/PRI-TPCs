Abstract:

Para este trabalho de casa final, foi proposta a criação de um servidor em NodeJS, que disponibilizasse uma API que permitisse a leitura e upload de ficheiros para a base de dados em MongoDB.

Assim sendo, apenas foi necessário criar uma rota ficheiros, que iria receber pedidos do tipo GET e POST, tanto para listar os ficheiros da BD como para inserir um ou mais ficheiros.

Por fim, foi utilizado JQuery para criar um formulário dinâmico para os ficheiros, assim como também foi criada uma função que apresenta um modal, cada vez que se clica num dos items da lista de ficheiros.
