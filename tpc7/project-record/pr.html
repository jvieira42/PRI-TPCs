Abstract:

Para este último trabalho de casa, foi proposta a criação de um servidor em NodeJS, que disponibilizasse uma API sobre os dados de um dataset de obras musicais. Esses dados foram disponibilizados em XML e convertidos para JSON, para poderem ser importados pela base de dados de MongoDB.

Era necessária a criação de duas rotas para o servidor poder responder a pedidos GET, as rotas obras e compositores.

Por fim, para obter a informação pretendida realizou-se as querys em MongoDB do controller, em que no caso da rota obras, tratam-se de querys de inversão estrutural.
