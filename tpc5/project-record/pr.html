Abstract:

Para este trabalho foi nos proposto a criação de um servidor em NodeJS utilizando a framework de desenvolvimento web Express, introduzida durante a aula da disciplina.

O tema requerido consistia na gestão de uma lista de alunos, das respetivas informações e notas de cada um.

Era pretendido a criação de rotas para que o servidor respondesse a pedidos GET, POST e DELETE, de modo a satisfazer as funcionalidades de listagem, inserção/remoção de alunos e notas. Também foi necessário criar os scripts de remoção. Toda esta informação é armazenada em formato JSON.

Finalmente, para representar e ver toda a informação foi utilizado o motor de templates Pug, para gerar dinamicamente as views do servidor. O estilo das páginas foi definido com o auxílio da stylesheet w3.css.
